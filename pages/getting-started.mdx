# Getting Started

## Instration

Inside your React project directory, run the following:

```sh
yarn add zenn-mde
```

Or with npm:

```sh
npm install zenn-mde
```

For the quick start, You should install the following packages as well

```sh
yarn add zenn-content-css zenn-markdown-html
```

## Quick Start

You can import `Editor` component as follows


```js
import { useState } from "react"
import { Editor } from "zenn-mde"

const defaultValue = "# Zenn MDE";

export const EditorDemo = () => {
  const [value, setValue] = useState(defaultValue);

  const handleValueChange = (newValue) => {
    setValue(newValue);
  }

  return (<Editor 
    previewClassName="znc"
    value={value} 
    onChange={handleValueChange} 
  />)
}
```
