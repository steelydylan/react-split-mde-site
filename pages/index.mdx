# React Split MDE

React Split MDE is a next generation markdown editor with high performance used at https://zenn.dev/

<EditorDemo />

## Overview


```js
import { useState } from "react";
import { Editor } from "react-split-mde";
import "react-split-mde/css/editor.css";

const defaultValue = "# React Split MDE";

export const EditorDemo = () => {
  const [value, setValue] = useState(defaultValue);

  const handleValueChange = (newValue) => {
    setValue(newValue);
  }

  return (<Editor 
    previewClassName="znc"
    value={value} 
    parser={markdownHTML}
    onChange={handleValueChange} 
  />)
}
```

In this example, `Editor` component accepts `parser` function which converts `markdown` into html.
`zenn-markdown-html` module is very useful as the `parser` function.

## Features

- customizable key bindings
- having useful event emitter to change text contents
