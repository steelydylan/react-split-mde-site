## Overview


```js
import { useState } from "react"
import { Editor } from "react-split-mde"
import "react-split-mde/css/index.css"

const defaultValue = "# React Split MDE"

export const EditorDemo = () => {
  const [value, setValue] = useState(defaultValue)

  const handleValueChange = (newValue) => {
    setValue(newValue)
  }

  return (<Editor 
    value={value}
    onChange={handleValueChange} 
  />)
}
```

In this example, `Editor` component accepts `parser` function which converts `markdown` into html.
`zenn-markdown-html` module is very useful as the `parser` function.
